Introduction to python-based Deep Learning tools

Published: Sun 10 January 2016
By Adriano Gil

In blog.

tags: deep-learning

Translations: pt

The very first steps to develop some Deep Learning experiment is to search a proper 3rd party component to handle the internals of such methods, or I mean, if you don't want to code everything from scratch.

In python, there are a lot of frameworks and libraries to work with Deep Learning. After I had participated in a course taught by Laurent Dihn from University of Montreal, I was introduced to the combined toolkit of Theano, Blocks and Fuel.

Theano is a very known mathematical python library used for deep learning. It allows to easily write experiments and evaluate them on GPU without difficulties. .. Explain Blocks

Blocks is a set of utilitary scripts that automate some tasks when using Theano.

Fuel is data processing library that handles some neat things like download and parse datasets.

Installing on Windows

I found some useful tips on this reddit post and I have tested it on Windows 8.1 and Windows 10.

The first step is download and install the Windows version of Anaconda. After a simple installation wizard, we need to install the required libraries for Theano. Thanksfully to Anaconda, we can use a very simple command line tool conda in the cmd-like Anaconda Prompt:

conda install pip six nose numpy scipy graphviz

Some needed python libraries are pre-requisite to specific Theano functions, like drawing debug graphs of operation nodes, and can also be used in Blocks or Fuel.

pip install pydot2 progressbar2 picklable-itertools

Theano can be installed using pip:

pip install theano

Fuel can be downloaded directly from github repository: https://github.com/mila-udem/fuel. In order to install it, we need to run the script setup.py:

cd fuel
python setup.py install

Fuel can be downloaded directly from github repository: https://github.com/mila-udem/blocks. In order to install it, we need to use python to execute the setuptools script setup.py

cd blocks
python setup.py install

Installing on Ubuntu

Some installation steps are published on Theano site, so I easily managed to install it on Ubuntu 14.10.

Download and e install all requisites of the libraries using the straightfoward command apt-get:

sudo apt-get install python-numpy python-scipy python-dev python-pip python-nose g++ libopenblas-dev git gfortran libhdf5-dev graphviz

Then Theano can be installed by pip:

pip install Theano

Fuel library can be also installed using pip directly from github repository:

pip install git+git://github.com/mila-udem/fuel.git

Blocks library follows the same command:

pip install git+git://github.com/mila-udem/blocks.git

Installing a Python IDE

After some good experience using PyDev and Eclipse, I recomend this light version Liclipse: http://www.liclipse.com/download.html. Some other good options can be found in: http://pedrokroger.net/choosing-best-python-ide/.
