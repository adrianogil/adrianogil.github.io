LiveOps Project: Popup Tool 2022

A LiveOps/Monetization tool project to show dynamically popups in Wildlife Studios games. Whenever a product manager wanted to deploy a new offer in game, it would only update remotely the popup html.

My role: I implemented an Unity3D SDK that allowed games to show dynamic popups. This SDK was incorporated in games such as Zooba, War Machines, Sniper3D.

Platforms: Android, iOS
